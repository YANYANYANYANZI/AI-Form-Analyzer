# 国家级区域环境与经济发展评价规范 (内部知识库)

## 1. 核心评估指标定义 (Definitions)
- **绿色发展指数 (Green Development Index, GDI)**：该指标是评估区域经济与环境协调发展的绝对核心。
  - **计算公式**：`绿色发展指数 = GDP_亿元 / 工业排放量_万吨 * 10000`
  - **业务解释**：该指数值越高，代表单位工业排放所带来的经济附加值越高，该省份的产业结构越属于“绿色低碳型”。

## 2. 数据审计与清洗红线 (Data Cleaning Redlines)
在对历年数据进行任何维度的统计、求平均或可视化之前，必须严格执行以下三条数据清洗红线：
1. **状态剔除**：凡是【备注】列内容为“待核查”或“数据异常”的记录，均属于未通过审计的无效数据，必须整行剔除。
2. **缺失值处理**：如果【工业排放量_万吨】存在缺失值 (NaN) 或空值，严禁使用均值填充，必须直接删除该行记录。
3. **逻辑异常值剔除**：任何【GDP_亿元】小于或等于 0 的记录均为系统统计故障，必须剔除。

## 3. 集团级可视化渲染规范 (Visualization Standards)
为了保障环保业务报表的严肃性，所有的 Agent 图表渲染必须遵守以下 UI 规范：
- **图表配色限制**：展示“绿色发展指数”或“环保排名”相关图表时，禁止使用带有警示意味的红色系（Reds）或默认的红蓝配色。**必须且只能使用绿色系渐变配色（如 Matplotlib 中的 `Greens` 或 `summer` color map）。**
- **数据展示数量**：在展示全国各省排名时，为了避免图表过于拥挤，默认只展示“排名前 10”或“排名后 10”的省份，并在条形图的柱子上标明具体数值。

## 4. 预警分析话术 (Alert Protocol)
- 在输出分析总结时，如果某省份计算出的【绿色发展指数】低于 10000，必须在分析结论中特别添加一行：`⚠️ [预警] 该地区产业结构偏重工业，单位排放经济效益低，建议加快产能转型。`
